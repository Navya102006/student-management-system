# Student Management System

A web-based Student Management System developed using Node.js, Express.js, MongoDB, and HTML/CSS/JavaScript.  
The system provides role-based login for Admin and Students to manage and view student information.

## Team Collaboration

This project is developed collaboratively using GitHub branches and pull requests.

Backend Developer:
- Node.js and Express backend development
- Authentication and authorization
- Database models and API logic

Frontend Developer:
- UI development using HTML, CSS, and JavaScript
- Dashboards and forms
- Styling and user interaction

## Features

- Role-based login (Admin and Student)
- Admin dashboard to manage students
- Student dashboard to view details
- Add, update, and delete student records
- Attendance and marks management
- MongoDB database integration
- GitHub collaboration using branches and pull requests

## How to Run

1. Clone the repository
2. Install backend dependencies using npm install
3. Start the server using node server.js
